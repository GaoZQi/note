# mkdocs

1. 安装 `pip install mkdocs`
2. 常用指令

    + `mkdocs new <work-space-name> #创建工作区`
    + `mkdocs serve #启动服务`
    + `mkdocs build #构建静态文件`
    + `mkdocs gh-deploy #部署到github`
    + `mkdocs -h    #查看帮助`

# mkdocs-material

1. 安装 `pip install mkdocs-material`
2. 配置 `mkdocs.yml`

    ```yml
    site_name: mkdocs-material
    theme: material
    ```

3. 配置 `mkdocs.yml` 侧边栏
